- **考核方式**：实验报告

## 教学目的
> 本课程是地理信息科学的应用实践补充内容，主要讲授GIS实验操作技术。目的让学生掌握一门功能强大的GIS软件，能够熟练地进行数据预处理、空间分析和建模。课程主要在机房上课，以[ArcGIS Desktop](https://www.arcgis.com/features/index.html)实验操作为主。

本课程分A、B班，授课内容相同。

主要内容：

- 第一章：ArcGIS简介
	> 详细介绍ArcGIS模块，常用空间数据文件格式以及文件转换
- 第二章：地图投影与变换
